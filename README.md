
# Library Management System

A full-stack Library Management System built using Java, Spring Boot, MySQL, JPA, Hibernate, and REST APIs.

## Features

* User registration and login with JWT authentication
* Role-based access (Admin / User)
* Book management (add, update, delete, search)
* Borrow and return books
* Reservation system for unavailable books
* Fine calculation for overdue returns
* Email notifications for due dates and returns
* Exception handling and validation
* Swagger API documentation

## Tech Stack

* Java 17
* Spring Boot
* Spring Security
* Spring Data JPA
* Hibernate
* MySQL
* Maven
* JWT
* Swagger
* Postman

## API Modules

* Authentication APIs
* Book APIs
* Borrow APIs
* Reservation APIs
* Admin APIs
